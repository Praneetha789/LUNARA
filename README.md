## LUNARA - YOUR PERSONAL VOICE ASSISTANT
LUNARA is an AI -powered **voice assistant** that can listen,recognize speech,and respond intelligently - build  completly from scrach using **python** and**web Technologies**
The goal of this project is to create a smart, interactive assistant that helps users perform tasks through natural voice interction
---
## FEATURES
**Speech Recognition** Understands voice commands in real time
**Smart Responses** Replies using natural speech
**Web Integration** Opens websites,performs searches, and execute basic tasks
**Learning-Based** Can be expanded with AI models for future improvement
**Custom-UI** Clean and minimal inteface built fot user friendliness
